# Logic Gates using ESP32 (NAND & NOR)

## Overview

In this project, I implemented basic logic gates using ESP32. Two push buttons are used as inputs and LEDs are used to show the output.

## Implementation

* Used buttons as input signals
* Implemented NAND and NOR logic
* Used LEDs to display output

## Working

* NAND: LED stays ON except when both buttons are pressed
* NOR: LED turns ON only when both buttons are not pressed

## Learning

* Understood how logic gates work in real hardware
* Learned to handle INPUT_PULLUP and inverted logic
* Connected digital logic concepts with practical implementation

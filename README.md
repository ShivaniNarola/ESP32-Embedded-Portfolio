# ESP32 Embedded Portfolio

This repository contains my hands-on practice with the ESP32 microcontroller. I am using this platform to build a strong understanding of embedded systems by implementing core concepts step by step.

The ESP32 is a 32-bit microcontroller with built-in Wi-Fi and Bluetooth, commonly used in IoT and real-time applications. It offers flexible GPIOs and multiple communication interfaces, making it suitable for practical embedded development.

## What I am working on

* Learning and implementing core microcontroller concepts
* Building small projects to understand real hardware behavior
* Improving debugging and wiring skills

## Concepts Covered

* GPIO (Input/Output control)
* PWM (Signal control and modulation)
* Timers and Interrupts
* Communication protocols: UART, SPI, I2C

## Approach

* Each concept is implemented through a small project
* Focus on both code and hardware connections
* Understanding how theory works in real systems

## Goal

To gradually move from basic concepts to more advanced embedded system implementations and build a strong practical foundation in embedded systems.
